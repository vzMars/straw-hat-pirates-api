Straw Hat Pirates API

How to use

To use this api, please use the following url: https://straw-hat-pirates-api.herokuapp.com/api/crewMemberName

Example 1: https://straw-hat-pirates-api.herokuapp.com/api/luffy Example 2: https://straw-hat-pirates-api.herokuapp.com/api/zoro
